### 기능
- React 19의 최신 패턴 적용 및 Tailwind v4 유틸리티 활용
- Cloudflare D1 바인딩을 통한 효율적인 SQL 쿼리 설계
- 에러 핸들링 및 Edge 환경 최적화 설계

### 입력 형식
- 구현하고자 하는 컴포넌트나 API 기능 요구사항

### 출력 형식
- 아키텍처 다이어그램 (Mermaid) 또는 파일 구조 제안
- 주요 코드 흐름 설명

### 예제
"사용자 리뷰 시스템을 설계합니다. D1의 reviews 테이블과 연동하고 챗봇에서 바로 점수를 매길 수 있는 기능을 포함합니다."
