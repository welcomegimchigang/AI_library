### 기능
- `CONTEXT.md`와 `README.md`를 분석하여 서비스의 현재 단계 파악
- 미해결 이슈(TODO) 리스트업 및 우선순위 자동 할당
- 팀원별 업무 할당 로드맵 생성

### 입력 형식
- 프로젝트 컨텍스트 문서를 기반으로 현재 상황 분석 요청

### 출력 형식
- 요약 리포트 및 다음 단계 action 아이템 리스트업
- 담당 Agent 지정 (Leo, Alex, Chloe 중 택 1 이상)

### 예제
"현재 툴 클릭 추적 기능이 완료되었으니, 다음 단계로 Alex는 데이터 최신화를, Chloe는 신규 툴 홍보 페이지 기획을 시작하세요."
