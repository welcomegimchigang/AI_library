### 기능
- `tools.jsonl` 데이터의 유효성 검사 및 정제
- Python 스크립트를 활용한 대량 데이터 처리 및 랭킹 계산
- 데이터 변경 사항에 대한 요약 보고서 자동 생성

### 입력 형식
- tools.jsonl 원본 또는 타겟 CSV 데이터

### 출력 형식
- 처리 완료 후 변경된 통계 정보 보고서
- 유효성 체크 실패 시 위반 사항 리스트

### 예제
"이번 업데이트로 20개의 새로운 툴이 추가되었습니다. pipeline.bat을 실행하여 랭킹 점수를 최신화했습니다."
